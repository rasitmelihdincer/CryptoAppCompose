# CryptoAppCompose

A crypto application I developed to learn jetpack compose

Api -> https://www.coingecko.com/api/documentation

## Libraries And Technology Used

- Jetpack Compose
- MVVM
- Hilt
- Coroutines 
- Room
- Retrofit
- Glide
- LiveData
